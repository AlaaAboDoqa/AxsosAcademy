## Project Overview

This repository contains the practical exercises, assignments, and test automation scripts developed during the QA Automation Bootcamp at Axsos Academy. The main objective of this project is to learn and implement industry-standard web automation practices, focusing on UI interactions, element locators, and automated test scenarios.

---

## Technical Stack

* **Language:** Java
* **Automation Framework:** Selenium WebDriver
* **Project Management:** Maven (via `pom.xml`)
* **Driver Management:** WebDriverManager (Boni Garcia)
* **IDE:** Eclipse IDE

---

## Core Features & Scripts

The project includes specialized Java classes designed to practice distinct aspects of Web Automation:

* **`HelloSelenium.java`** – Initial configuration, setting up GeckoDriver, and initializing the Firefox browser instance.
* **`WebElementsPractice.java`** – Interacting with fundamental UI elements like input fields, buttons, and checkboxes.
* **`XPathPractice.java`** – Mastering dynamic locator strategies using custom and absolute/relative XPath expressions.
* **`SelectUI.java`** – Handling static and dynamic dropdown menus using the Selenium Select class (selecting by index, value, and visible text).
* **`OnlyTesting.java`** – A dedicated sandbox class for debugging code snippets and trying out experimental scripts.
